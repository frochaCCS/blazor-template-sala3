# Security Policy

We take security seriously and welcome responsible disclosure of vulnerabilities.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

**Do not file public issues for security reports.**

If you discover a security vulnerability in this project, please report it via:

- **GitHub Security Advisories**: Use the "[Report a vulnerability](https://github.com/frochaCCS/ITSupportDesk/security/advisories)" feature in the Security tab of this repository.
- **Private message**: Contact the maintainer(s) listed in [CODEOWNERS](.github/CODEOWNERS).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential workarounds or mitigations

We aim to acknowledge reports within 48 hours and will work with you to resolve critical issues promptly.
