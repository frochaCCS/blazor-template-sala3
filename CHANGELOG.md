## [0.1.0] - 2026-05-22

### Added
- Initial ITSupportDesk template scaffold.
- Blazor Web App foundation with ASP.NET Identity authentication.
- Support ticket management system with admin and user roles.
- Comment system for ticket collaboration.
- E2E test suite with Playwright and xUnit.
- GitHub Actions CI pipeline (baseline).
- Repository meta files (LICENSE, SECURITY.md, CONTRIBUTING.md, CHANGELOG.md).
